# HowToCook: 编程风格的烹饪指南

## 简介

HowToCook 是一个开源项目，将软件工程原则应用于烹饪食谱。由 Anduin2017 创建，它以结构化、无歧义的格式呈现食谱，消除了传统烹饪书中常见的模糊指令。

该项目使用精确的计量、分步算法和清晰的组织，使任何能遵循指令的人都能轻松烹饪。

## 为什么结构化食谱很重要

传统食谱常常使用不精确的语言，导致结果不一致。

| 传统指令 | 问题 | HowToCook 方法 |
|---------|------|---------------|
| "加点盐" | 主观 | "加 5g 盐" |
| "煮到熟" | 模糊 | "180C 煮 8 分钟" |
| "一小撮调料" | 不精确 | "加 2g 调料" |
| "中火" | 含糊 | "炉灶设为 10 档中的 5 档" |
| "静置一会儿" | 未定义 | "静置 5 分钟" |

## 食谱结构

HowToCook 中的每个食谱都遵循标准化格式。

| 部分 | 内容 | 用途 |

## 基础食谱

### 白米饭

**食材：**
| 食材 | 数量 |
|------|------|
| 米 | 300g |
| 水 | 360ml |

**步骤：**
1. 用冷水冲洗大米直到水变清（3-4 次）
2. 将米和水加入锅中
3. 大火烧开（约 5 分钟）
4. 转小火，盖上锅盖
5. 煮 15 分钟
6. 关火，静置 5 分钟
7. 用叉子拨松即可

**备注：**
| 因素 | 建议 |
|------|------|
| 米种 | 茉莉香米或长粒米 |
| 水比 | 水米比 1.2:1 |
| 静置时间 | 不可跳过 |

### 炒蔬菜

**食材：**
| 食材 | 数量 |
|------|------|
| 混合蔬菜 | 400g |
| 植物油 | 15ml |
| 蒜（切碎） | 10g |
| 酱油 | 15ml |
| 盐 | 3g |
| 糖 | 2g |

**步骤：**
1. 洗净蔬菜，切成均匀大小
2. 大火热锅，加油 1 分钟
3. 加蒜，翻炒 30 秒
4. 加蔬菜，翻炒 3-5 分钟
5. 加酱油、盐和糖
6. 再翻炒 1 分钟
7. 立即上桌

**时间：**
| 步骤 | 时长 |
|------|------|
| 准备 | 10 分钟 |
| 烹饪 | 7 分钟 |
| 总计 | 17 分钟 |

### 蛋炒饭

**食材：**
| 食材 | 数量 |
|------|------|
| 米饭（冷饭） | 400g |
| 鸡蛋 | 3 个 |
| 植物油 | 30ml |
| 酱油 | 15ml |
| 盐 | 2g |
| 葱花 | 30g |

**步骤：**
1. 鸡蛋加 1g 盐打散
2. 葱切碎
3. 大火热锅 1 分钟
4. 加 15ml 油，晃匀锅底
5. 倒入鸡蛋，翻炒 30 秒
6. 盛出鸡蛋备用
7. 加剩余 15ml 油
8. 加米饭，打散结块，翻炒 2 分钟
9. 加回鸡蛋，翻炒 1 分钟
10. 加酱油，翻炒 30 秒
11. 加葱花，翻炒 30 秒
12. 立即上桌

**备注：**
| 技巧 | 原因 |
|------|------|
| 用冷饭 | 防止粘糊 |
| 大火 | 产生锅气 |
| 快速操作 | 防止过熟 |

## 中级食谱

### 麻婆豆腐

**食材：**
| 食材 | 数量 |
|------|------|
| 嫩豆腐 | 400g |
| 猪肉末 | 150g |
| 豆瓣酱 | 30g |
| 蒜（切碎） | 15g |
| 姜（切碎） | 10g |
| 花椒粉 | 2g |
| 酱油 | 10ml |
| 水淀粉 | 15g 淀粉 + 30ml 水 |
| 植物油 | 20ml |
| 葱花 | 20g |

**步骤：**
1. 豆腐切成 2cm 方块
2. 烧开水，焯豆腐 2 分钟，沥干
3. 中大火热锅加油
4. 加猪肉末，炒至变色（3 分钟）
5. 加豆瓣酱，炒 1 分钟至出香
6. 加蒜和姜，炒 30 秒
7. 加 200ml 水和酱油，烧开
8. 轻轻加入豆腐，小火煮 5 分钟
9. 搅拌水淀粉，倒入锅中
10. 轻轻搅拌至酱汁变稠（1 分钟）
11. 撒花椒粉
12. 撒葱花点缀

**时间：**
| 步骤 | 时长 |
|------|------|
| 准备 | 10 分钟 |
| 焯水 | 3 分钟 |
| 烹饪 | 12 分钟 |
| 总计 | 25 分钟 |

## 高级食谱

### 红烧肉

**食材：**
| 食材 | 数量 |
|------|------|
| 五花肉 | 500g |
| 冰糖 | 30g |
| 酱油 | 30ml |
| 老抽 | 15ml |
| 绍兴酒 | 30ml |
| 姜 | 3 片 |
| 八角 | 2 个 |
| 桂皮 | 1 段 |
| 水 | 500ml |

**步骤：**
1. 五花肉切成 3cm 方块
2. 焯水 3 分钟，沥干
3. 干锅小火融化冰糖至焦糖色
4. 加猪肉，翻裹焦糖
5. 加姜、八角、桂皮，炒 1 分钟
6. 加酱油、老抽、绍兴酒
7. 加水，烧开
8. 转小火，盖盖，炖 90 分钟
9. 开盖，大火收汁（10 分钟）
10. 酱汁浓稠发亮即可

**时间：**
| 步骤 | 时长 |
|------|------|
| 准备 | 15 分钟 |
| 烹饪 | 105 分钟 |
| 总计 | 120 分钟 |

## 常见错误与解决方案

| 错误 | 原因 | 解决方案 |
|------|------|---------|
| 米饭太烂 | 水太多 | 减少水比 |
| 蒜炒糊 | 火太大 | 油稍凉后再加蒜 |
| 蔬菜出水 | 锅太挤 | 分批炒 |
| 味道淡 | 调味不够 | 尝味调整 |
| 肉太老 | 过度烹饪 | 使用温度计 |
