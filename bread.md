# The Bread Code 教程：手工面包烘焙

## 简介

The Bread Code 是一份使用科学原理在家烘焙手工面包的综合指南。本教程涵盖了面包制作的基础知识，从理解原料到掌握各种技术，以获得稳定、高质量的结果。

## 基本原料

### 面粉类型

| 面粉 | 蛋白质含量 | 最佳场景 | 吸水率 |
|------|-----------|---------|--------|
| 高筋面粉 | 12-14% | 手工面包、法棍 | 65-70% |
| 中筋面粉 | 10-12% | 通用烘焙 | 60-65% |
| 全麦面粉 | 13-14% | 全谷物面包 | 70-80% |
| 黑麦面粉 | 8-10% | 黑麦面包、酸面团 | 80-90% |
| 斯佩尔特面粉 | 12-14% | 坚果风味面包 | 60-65% |
| 粗粒面粉 | 12-13% | 意大利面包、意面 | 60-65% |

### 原料功能

| 原料 | 功能 | 典型比例 |
|------|------|---------|
| 面粉 | 结构（面筋） | 100%（基准） |
| 水 | 水合、面筋发展 | 60-85% |
| 盐 | 风味、面筋强度 | 1.8-2.2% |
| 酵母 | 膨松（发酵） | 0.5-2% |
| 酸面团酵种 | 天然膨松 | 10-20% |
| 糖 | 酵母食物、褐变 | 0-5% |
| 油脂（黄油/油） | 柔软、风味 | 0-10% |

## 面包师百分比

### 原理

所有原料按占面粉总重量的百分比计算。

| 原料 | 面包师百分比 | 示例（1000g 面粉） |
|------|-------------|-------------------|
| 面粉 | 100% | 1000g |
| 水 | 70% | 700g |
| 盐 | 2% | 20g |
| 酵母 | 1% | 10g |
| **合计** | 173% | 1730g |

### 常见水合度

| 水合度 | 面团手感 | 结果 |
|--------|---------|------|
| 55-60% | 硬，易操作 | 紧密组织 |
| 60-65% | 结实，可控 | 中等组织 |
| 65-70% | 柔软，略粘 | 开放组织 |
| 70-80% | 非常粘，有延展性 | 非常开放的组织 |
| 80%+ | 极湿 | 恰巴塔风格 |

## 酸面团酵种

### 创建酵种

| 天数 | 操作 | 喂养比例 |
|------|------|---------|
| 1 | 混合 50g 面粉 + 50g 水 | 1:1:1 |
| 2 | 弃掉一半，喂养 50g 面粉 + 50g 水 | 1:1:1 |
| 3-6 | 每天重复弃掉和喂养 | 1:1:1 |
| 7+ | 当体积翻倍时喂养（6-12 小时） | 1:5:5 |

### 酵种就绪测试

| 测试 | 方法 | 结果 |
|------|------|------|
| 浮水测试 | 放一勺到水中 | 浮起 = 就绪 |
| 体积翻倍 | 体积增加 | 4-8 小时内翻倍 |
| 气泡 | 表面和侧面 | 活跃冒泡 |
| 气味 | 香气检查 | 愉悦，微酸 |
| 峰值 | 最高点 | 略过峰值 |

### 酵种维护

| 计划 | 喂养比例 | 存储 |
|------|---------|------|
| 每天烘焙 | 1:5:5 | 室温 |
| 每周烘焙 | 1:5:5 | 冰箱，每周喂养 |
| 每月烘焙 | 1:1:1 | 冰箱，每月喂养 |

| 步骤 | 时间 | 时长 | 操作 |
|------|------|------|------|

## 混合技术

### 拉伸折叠

| 步骤 | 操作 | 时机 |
|------|------|------|
| 1 | 湿手，抓住面团边缘 | 每 30 分钟 |
| 2 | 向上拉伸 | 基础发酵期间 |
| 3 | 折到中心 | 共 4-6 组 |
| 4 | 旋转 90 度 | |
| 5 | 重复 3 次 | |

### 拍打折叠

| 步骤 | 操作 |
|------|------|
| 1 | 抓住面团，从台面提起 |
| 2 | 拍到台面上 |
| 3 | 折叠到自身上 |
| 4 | 旋转并重复 |
| 时长 | 5-10 分钟直到光滑 |

### Rubaud 方法

| 步骤 | 操作 |
|------|------|
| 1 | 湿手，抓取一部分面团 |
| 2 | 提起并拍回碗中 |
| 3 | 稍微旋转碗 |
| 4 | 重复 5 分钟 |

## 基础发酵

### 正确发酵的标志

| 指标 | 说明 |
|------|------|
| 体积增加 | 增加 50-75% |
| 气泡 | 表面和侧面可见 |
| 晃动 | 摇晃时面团抖动 |
| 圆顶 | 顶部略微隆起 |
| 光滑 | 表面变得光滑 |
| 香气 | 愉悦的酵母味 |

### 温度指南

| 温度 | 发酵时间 | 结果 |
|------|---------|------|
| 22-24°C (72-75°F) | 4-6 小时 | 平衡风味 |
| 25-28°C (77-82°F) | 3-4 小时 | 更快，更温和 |
| 18-20°C (64-68°F) | 8-12 小时 | 更慢，更复杂 |
| 15°C (59°F) | 16-24 小时 | 非常慢，复杂 |

## 整形

### 圆形（Boule）

| 步骤 | 操作 |
|------|------|
| 1 | 将面团翻到撒粉台面 |
| 2 | 将边缘折向中心 |
| 3 | 翻面，接缝朝下 |
| 4 | 双手环抱面团 |
| 5 | 向身体方向拉以产生张力 |
| 6 | 重复直到紧绷 |

### 椭圆形（Batard）

| 步骤 | 操作 |
|------|------|
| 1 | 将面团压平成长方形 |
| 2 | 将上三分之一向下折叠 |
| 3 | 将下三分之一向上折叠 |
| 4 | 从顶部卷起，封口 |
| 5 | 封好两端和接缝 |
| 6 | 接缝朝上放入发酵篮 |

## 常见问题

| 问题 | 原因 | 解决方案 |
|------|------|---------|
| 组织紧密 | 发酵不足 | 延长基础发酵 |
| 形状扁平 | 发酵过度或面粉筋度弱 | 减少时间或使用高筋面粉 |
| 质地发粘 | 未烤熟 | 延长烘烤，检查内部温度 |
| 外皮苍白 | 无蒸汽或温度低 | 添加蒸汽，提高温度 |
| 大孔洞 | 整形问题 | 改进整形技术 |
| 酸味过重 | 发酵时间过长 | 减少发酵时间 |
| 无烤箱膨胀 | 酵种失活或发酵不足 | 检查酵种，调整发酵 |

## 总结

手工面包烘焙是科学与工艺的结合。理解水合度、发酵和技术可以带来稳定的结果。从简单的酸面团配方开始，掌握基础，然后尝试不同的面粉、水合度和整形技术。
