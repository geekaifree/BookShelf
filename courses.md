# CS 视频课程: 精选计算机科学学习资源

## 简介

CS 视频课程是来自全球大学和机构的计算机科学视频课程精选列表。该仓库按学科领域组织教育视频内容，方便查找高质量课程用于自学或学术参考。

该集合涵盖计算机科学从基础到高级的主题，全部可在线免费获取。

## 为什么使用视频课程？

## 课程选择指南

### 按经验级别

| 级别 | 推荐课程 |
|------|---------|
| 完全初学者 | CS50, 6.0001, CS 61A |
| 有一些编程经验 | 6.006, Algorithms Part I, CS 61B |
| 中级 | CS 162, CS 186, CS 221 |
| 高级 | 6.829, 6.830, CS 285 |

### 按学习目标

| 目标 | 推荐路径 |
|------|---------|
| Web 开发 | CS50 → CS 61A → CS 169 |
| 数据科学 | 6.0001 → CS 61A → CS 229 |
| 系统编程 | CS 61A → CS 61B → CS 162 |
| AI 研究 | CS 229 → CS 231n → CS 285 |
| 网络安全 | CS 61A → CS 161 → 6.858 |

### 按时间投入

| 可用时间 | 推荐方式 |
|---------|---------|
| 2-3 小时/周 | 仅观看讲座 |
| 5-10 小时/周 | 讲座 + 阅读 |
| 15-20 小时/周 | 完整课程含作业 |
| 40+ 小时/周 | 多门课程 |
