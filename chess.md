# 国际象棋：学习指南

## 棋子动作

|件|运动|特别规则 |
|-------|----------|---------------|
|国王|任意方向一个正方形 |无法进入检查；易位 |
|女王 |任意方向任意数量的正方形 |最强大的一块|
|车 |水平或垂直任意数量的正方形 |易位伙伴 |
|主教|任意数量的对角方格|每个主教都保持一种颜色 |
|骑士| L型（2+1方格） |可以跳过其他棋子|
|典当|前进一格（从起始位置开始两格）|沿对角线捕捉；路过；促销|

### 特殊动作

- **易位**：国王向车移动两个方格，然后车跳到另一边。条件：两颗棋子均未移动，两颗棋子之间没有棋子，K 不在受控状态且未通过检查。
- **En passant**：前进两格的棋子可以被相邻敌方棋子捕获，就好像它移动了一格一样。必须立即完成。
- **晋升**：到达相反后排的棋子晋升为任何棋子（通常是皇后）。

## 基本策略

战术是获取材料或将死敌人的一系列简短的动作。认识战术模式是最快的改进方法。

### 叉子

一枚棋子同时攻击两个或多个敌方棋子。对手只能救一个。

- 骑士叉子特别危险，因为骑士可以攻击无法反击的棋子。
- 敌人第四或第五排的棋叉很常见，但经常被忽视。

### 引脚

棋子攻击敌方棋子，敌方棋子无法移动，除非暴露其后面更有价值的棋子。

- **绝对钉子**：被钉住的棋子正在保护国王——它在法律上不能移动。
- **相对固定**：固定的棋子正在屏蔽有价值的棋子（例如皇后），但可以合法移动。

### 串烧

针的反面。有价值的棋子受到攻击并被迫移动，从而暴露出其后面的价值较低的棋子以供捕获。

### 发现的攻击

移动一个棋子会显示其后面另一棋子的攻击。移动的棋子也可以造成威胁，产生双重攻击。

- **发现检查**：发现的攻击进行检查。极其强大，因为对手必须应对检查，让被移动的棋子自由行动。

### 偏转和诱饵

- **偏转**：迫使防守者偏离其职责。
- **诱饵**：将棋子引诱到易受攻击的方格。

### 重载

一个棋子承担了太多的防守职责。你可以通过攻击它所防御的事物之一来利用这一点，迫使它放弃另一个事物。

## 开放原则

1. **控制中心**：中心的四个方格（e4，d4，e5，d5）是最重要的。放置在这里的棋子具有最大的移动性。
2. **开发棋子**：尽早让骑士和主教出局。没有充分的理由，不要将同一个棋子移动两次。
3. **国王安全**：城堡早期，通常在前10步之内。
4. **不要提前带出蚁后**：它会成为敌人发展的目标。
5. **连接车**：一旦开发出小棋子并且国王已经城堡，车就可以看到彼此。

## 残局基础知识

残局是赢得或挽救比赛的地方。在记住空缺之前先研究一下它们。

### 国王和兵的残局

- **反对派**：当国王面对面时，他们之间只有一个方格，不移动的一方就有反对派——这是推进的关键优势。
- **通过的棋子**：没有敌人棋子阻挡或守卫其晋级之路的棋子。推过去的棋子。
- **方格规则**：为了抓住经过的棋子，对方国王必须能够进入棋子晋升路径形成的“方格”。
- **关键方格**：对于第 5 排的棋子，其前面的第 6 排的三个方格是关键方格 - 如果友方国王占据其中任何一个方格，则该棋子晋级。

### Rook 残局

最常见的残局类型。

- **卢塞纳位置**：车+兵对车的获胜位置。学习“桥梁”技术。
- **Philidor位置**：车+兵对车的绘图技术。将车保持在第六位，然后切换到从后面检查。
- **活动比物质更重要**：活跃的车通常比兵更有价值。

### 残局基础知识

|情况|结果 |
|-----------|--------|
|国王 + 王后 vs 国王 |获胜（将死很简单）|
|国王 + 车 vs 国王 |获胜（迫使国王获胜，交出队友）|
|国王 + 2 主教 vs. 国王 |获胜（将国王逼至角落）|
|国王+主教+骑士VS国王|获胜（但技术困难）|
|国王+骑士VS国王|平局（不能强制将死）|
|国王 + 主教 vs. 国王 |平局（不能强制将死）|
|国王 + 2 骑士对国王 |平局（除非对手有棋子）|

## 学习方法

### 拼图训练

日常战术谜题是提高专家水平的最有效的训练方法。

|平台|成本|拼图品质 |
|----------|------|----------------|
|巫妖谜题 |免费|数据库大，难度自适应 |
| Chess.com 谜题 |免费增值 |品种齐全，主题套装 |
|国际象棋节奏|免费增值 |出色的标记和统计 |
| CT-ART |付费（移动）|经典、高度结构化|

每天花 15-30 分钟做谜题。注重准确性而不是速度。在移动之前充分计算变化。

### 游戏分析

1.玩游戏（学习时建议时间控制在15分钟以上）。
2. 首先自己分析——找出你认为哪里出了问题。
3. 使用引擎来检查您的分析。
4.专注于关键时刻，而不是每一步。
5. 写下每场比赛的一个教训。

## 时间管理

### 选择学习时间控制

|时间控制|适合 |笔记|
|-------------|----------|-------|
| 15+10（快速）|学习、分析|思考时间充足，还算实用|
| 30+0 或 60+30（经典）|深算、认真学习 |最适合改进但耗时 |
| 5+3（闪电战）|模式识别，有趣|不适合学习——速度太快，无法深入思考 |
| 3+0（闪电战）|娱乐 |避免用于研究目的 |

### 游戏中的时间管理

- 如果您不熟悉该职位，请在开头花更多时间。
- 将大部分时间用在做出关键决策的中间游戏上。
- 不要急于进入最后阶段——一个错误就可能改变结果。
- 如果你有明显的优势，不要立即玩——确认没有战术。
- 利用对手的思考时间来计划自己的行动。

|评级范围|重点领域 |
|-------------|-------------|
|初级（1000以下）|停止犯错（行动前检查），学习基本的将死 |
| 1000-1200|战术模式识别，基本开放原理|
| 1200-1500 |位置理解、残局基础知识、较长的比赛 |
| 1500-1800 |开局剧目、战略规划、算计深度|
| 1800-2000 |深入的残局研究、位置牺牲、心理弹性|
| 2000+ |专业的开局准备、复杂的残局、持续的练习 |
