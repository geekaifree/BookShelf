本指南涵盖了 macOS 上的必备开源应用，按类别组织。这些应用提供免费的商业软件替代方案，同时保持高质量和积极的开发。

## 包管理器

### Homebrew

| 命令 | 用途 |
|------|------|
| `brew install` | 安装 formula |
| `brew install --cask` | 安装应用 |
| `brew update` | 更新 Homebrew |
| `brew upgrade` | 升级包 |
| `brew cleanup` | 移除旧版本 |
| `brew list` | 列出已安装 |

### MacPorts

| 命令 | 用途 |
|------|------|
| `sudo port install` | 安装 port |
| `sudo port selfupdate` | 更新 MacPorts |
| `sudo port upgrade outdated` | 升级 ports |
| `port installed` | 列出已安装 |

## 总结

macOS 开源生态系统在所有应用类别中提供高质量的替代方案。Homebrew 简化了这些工具的安装和管理，而活跃的开源社区确保了持续的开发和支持。
