## 密码

### 密码最佳实践

| 实践 | 说明 |
|----------|-------------|
| 使用唯一密码 | 不要在不同网站重复使用密码 |
| 长密码 | 至少 16 个字符 |
| 随机生成 | 使用密码管理器生成 |
| 不含个人信息 | 避免姓名、生日、常见单词 |
| 定期更新 | 怀疑泄露时立即更改 |

### 密码强度

| 长度 | 熵值 | 强度 |
|--------|---------|----------|
| 8 个字符 | ~52 bits | 弱 |
| 12 个字符 | ~78 bits | 中等 |
| 16 个字符 | ~104 bits | 强 |
| 20 个字符 | ~130 bits | 很强 |
| 24 个字符 | ~156 bits | 极强 |

### 密码管理器

| 管理器 | 平台 | 开源 | 价格 |
|---------|-----------|-------------|-------|
| Bitwarden | 全平台 | 是 | 免费/高级 |
| 1Password | 全平台 | 否 | 付费 |
| KeePassXC | 桌面 | 是 | 免费 |
| Dashlane | 全平台 | 否 | 付费 |
| Proton Pass | 全平台 | 是 | 免费/付费 |

### 密码管理器设置

1. 选择信誉良好的密码管理器
2. 创建强主密码（20+ 字符）
3. 启用双因素认证
4. 导入现有密码
5. 为所有账户生成唯一密码
6. 安全存储恢复代码

### 主密码技巧

| 技巧 | 示例 |
|-----|---------|
| 使用密码短语 | "correct horse battery staple" |
| 增加复杂性 | 混入数字和符号 |
| 便于记忆 | 只有你知道的个人含义 |
| 记录下来 | 初始阶段离线安全存储 |

## VPN

### VPN 的作用

| 功能 | 说明 |
|---------|-------------|
| 加密流量 | 保护传输中的数据 |
| 隐藏 IP | 防止追踪 |
| 绕过地理限制 | 访问地区限制内容 |
| 公共 WiFi 安全 | 在不受信任的网络上安全 |

### VPN 不能做的事

| 误解 | 现实 |
|---------------|---------|
| 完全匿名 | 仍可通过其他方式追踪 |
| 阻止所有追踪 | Cookie、指纹仍有效 |
| 防止恶意软件 | 仍需杀毒软件 |
| 使非法变合法 | 法律仍然适用 |
