## 简介

AntennaPod 是一款免费、开源的 Android 播客管理和播放器。它提供完整的播客体验，没有广告、追踪或订阅费用。AntennaPod 支持订阅播客、下载剧集、变速播放和丰富的自定义功能。

与专有播客应用不同，AntennaPod 让你完全控制收听体验和数据。

## 为什么使用 AntennaPod？

| 特性 | AntennaPod | 专有应用 |
|------|-----------|---------|
| 费用 | 永久免费 | 通常需要订阅 |
| 广告 | 无 | 常见 |
| 追踪 | 无 | 大量 |
| 开源 | 是 | 否 |
| 数据导出 | 完全控制 | 有限 |
| 自定义 | 丰富 | 有限 |
| 离线播放 | 是 | 有时受限 |

## 核心功能

| 功能 | 描述 |
|------|------|

## 安装

### 从应用商店

| 商店 | 可用性 |
|------|-------|
| Google Play Store | 是 |
| F-Droid | 是 |
| GitHub Releases | APK 下载 |

### 从 F-Droid

1. 从 f-droid.org 安装 F-Droid
2. 搜索 "AntennaPod"
3. 点击安装
4. 打开应用

### 从 GitHub

1. 访问 github.com/AntennaPod/AntennaPod/releases
2. 下载最新 APK
3. 启用"允许未知来源"
4. 安装 APK

## 故障排除

| 问题 | 原因 | 解决方案 |
|------|------|---------|
| 剧集无法下载 | 网络问题 | 检查 WiFi 连接 |
| 播放停止 | 电池优化 | 为 AntennaPod 禁用 |
| 没有声音 | 音量/静音 | 检查设备音量 |
| 应用崩溃 | 数据损坏 | 清除应用缓存 |
| 同步不工作 | Gpodder 问题 | 重新认证 |
