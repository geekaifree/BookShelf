## VPN 服务

VPN 在访问在线内容时对隐私至关重要。

| 特性 | 考量 |
|---------|---------------|
| No-log policy | 提供商不跟踪活动 |
| Kill switch | VPN 断开时阻止流量 |
| Server count | 更多服务器意味着更好的性能 |
| Protocol support | WireGuard, OpenVPN, IKEv2 |
| Simultaneous connections | 允许的设备数量 |
