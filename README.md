## 🌟 项目简介

BookShelf 是一个 **多语言资源集合**，汇集了各领域的精选学习资料、工具和最佳实践。每个文档都经过精心整理，以中文呈现，帮助用户快速获取高质量资源。

- 🌍 **多领域覆盖** — 技术、金融、设计、生活等 15+ 个领域
- 📱 **移动友好** — Markdown 格式，随时随地阅读
- 🔍 **持续更新** — 定期补充最新资源与工具
- 🎓 **由浅入深** — 从入门到进阶的完整学习路径
- 📊 **表格展示** — 清晰易读，快速查找
