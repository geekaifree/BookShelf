## 合成器

合成器通过电子方式生成声音，提供丰富的音色和纹理。

### 合成类型

| 类型 | 方法 | 特点 | 示例 |
|------|------|------|------|
| 减法 | 过滤谐波丰富的信号 | 经典、温暖 | Moog、Minimoog |
| 加法 | 合成正弦波 | 精确、复杂 | Hammond organ |
| FM | 频率调制 | 金属感、钟声 | DX7、FM8 |
| 波表 | 扫描波形 | 演变、数字化 | Serum、Massive |
| 粒子 | 操纵微小音频粒子 | 纹理、实验性 | GrainSpace |
| 物理建模 | 模拟物理对象 | 原声、逼真 | Pianoteq、Strum |
| 模块化 | 灵活信号路由 | 实验性、无限制 | VCV Rack、Reaktor |

### 免费合成器

| 合成器 | 类型 | 平台 | 核心功能 |
|--------|------|------|----------|

| Surge | 混合 | 全平台 | 极其多变 |
| Dexed | FM | 全平台 | DX7 模拟器 |
| Helm | 减法 | 全平台 | 易于学习 |
| TAL-NoiseMaker | 减法 | 全平台 | 经典音色 |
| VCV Rack | 模块化 | 全平台 | Eurorack 模拟器 |

| OB-Xd | 减法 | 全平台 | Oberheim 模拟 |

### 合成器参数

| 参数 | 描述 | 效果 |
|------|------|------|
| 振荡器 | 声源 | 生成原始波形 |
| 滤波器 | 去除频率 | 塑造音色 |
| 包络 | 基于时间的控制 | 控制声音如何演变 |

| 放大器 | 音量控制 | 控制输出电平 |

## 混音

混音将歌曲的所有元素平衡为一个有凝聚力的立体声输出。

### 混音基础

| 元素 | 角色 | 频率范围 |
|------|------|----------|
| 底鼓 | 节奏基础 | 60-100 Hz |
| 贝斯 | 和声基础 | 80-250 Hz |
| 人声 | 旋律、歌词 | 300 Hz - 5 kHz |
| 军鼓 | 节奏强调 | 200 Hz - 5 kHz |
| 镲片 | 节奏纹理 | 5 kHz - 16 kHz |
| 吉他/键盘 | 和声、纹理 | 200 Hz - 8 kHz |
| Pad | 氛围 | 200 Hz - 8 kHz |

### 混音流程

| 步骤 | 操作 | 工具 |
|------|------|------|
| 1. 增益分级 | 设置合适电平 | 推子、微调 |
| 2. 声像 | 将声音放在立体声场中 | 声像旋钮 |
| 3. EQ | 为每个元素腾出空间 | 参数 EQ |
| 4. 压缩 | 控制动态 | 压缩器 |
| 5. 混响/延迟 | 创造深度和空间 | 混响、延迟 |
| 6. 自动化 | 随时间的动态变化 | 自动化轨道 |
| 7. 效果 | 创意处理 | 各种 |
| 8. 参考 | 与专业曲目比较 | 参考插件 |

### EQ 指南

| 频率范围 | 内容 | 常见调整 |
|----------|------|----------|
| 超低（20-60 Hz） | 超低音、底鼓底部 | 高通滤波大多数轨道 |
| 低（60-250 Hz） | 贝斯、底鼓主体 | 切除浑浊（200-300 Hz） |
| 中低（250-500 Hz） | 人声主体、吉他温暖 | 切除箱音 |
| 中（500 Hz - 2 kHz） | 人声存在感、军鼓主体 | 提升清晰度 |
| 中高（2-5 kHz） | 人声清晰度、吉他咬劲 | 提升存在感 |
| 高（5-10 kHz） | 镲片、人声空气感 | 提升亮度 |
| 空气（10-20 kHz） | 闪烁、气息 | 轻微提升 |

### 压缩设置指南

| 源 | 比率 | 启动 | 释放 | 备注 |
|----|------|------|------|------|
| 人声 | 3:1-4:1 | 中 | 中 | 温和、透明 |
| 鼓 | 4:1-8:1 | 快 | 快 | 有力 |
| 贝斯 | 4:1-6:1 | 中 | 中 | 平衡动态 |
| 吉他 | 2:1-4:1 | 中 | 中 | 温和控制 |
| 母线 | 2:1-3:1 | 慢 | 自动 | 粘合、微妙 |

## MIDI

MIDI（乐器数字接口）是传递音乐信息的协议。

### MIDI 基础

| 消息 | 描述 | 数据 |
|------|------|------|
| Note On | 按下键 | 音符号、力度 |
| Note Off | 松开键 | 音符号 |
| Control Change | 控制器移动 | 控制器号、值 |
| Program Change | 预设变化 | 音色号 |
| Pitch Bend | 弯音轮 | 弯音量 |
| Aftertouch | 按键后压力 | 压力值 |

### MIDI 音符号

| 音符 | 第 2 八度 | 第 3 八度 | 第 4 八度 | 第 5 八度 |
|------|-----------|-----------|-----------|-----------|
| C | 36 | 48 | 60 | 72 |
| D | 38 | 50 | 62 | 74 |
| E | 40 | 52 | 64 | 76 |
| F | 41 | 53 | 65 | 77 |
| G | 43 | 55 | 67 | 79 |
| A | 45 | 57 | 69 | 81 |
| B | 47 | 59 | 71 | 83 |

### MIDI 控制器

| 控制器类型 | 描述 | 价格范围 |
|-----------|------|----------|
| 键盘 | 钢琴式按键 | $50-2000 |
| Pad 控制器 | 鼓垫 | $50-500 |
| 控制面 | 推子、旋钮 | $100-2000 |
| 吹管控制器 | 吹奏乐器 | $200-1000 |
| 吉他控制器 | 吉他 MIDI | $200-800 |

### MIDI 编辑

| 操作 | 描述 |
|------|------|
| 量化 | 将音符对齐网格 |
| 力度 | 调整音符响度 |
| 长度 | 改变音符时值 |
| 移调 | 上下移动音符 |
| 人性化 | 添加轻微时间变化 |
| 琶音 | 从和弦创建琶音 |

## 总结

音乐制作是一项结合技术技能和艺术表达的创造性追求。关键要点：

- **从简单开始**：使用免费工具（LMMS、GarageBand）学习基础
- **精通你的 DAW**：深入了解一个 DAW 比浅尝多个要好
- **定期练习**：持续练习比偶尔的长时间练习更快建立技能
- **完成歌曲**：完成项目比开始很多项目学到更多
- **使用参考曲目**：将你的作品与专业音乐比较
- **投资监听**：好的耳机或音箱是必需的
- **学习音乐理论**：理解基础能提升你的音乐
- **协作**：与他人合作加速学习
