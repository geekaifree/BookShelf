## 简介

AdventureLog 是一款开源旅行记录应用，帮助旅行者记录冒险经历、整理照片、追踪统计数据并分享体验。它提供了一个全面的平台来保存旅行回忆。

### 主要功能

| 功能 | 描述 |
|------|------|
| 旅行日志 | 记录每次旅行的详细信息 |
| 照片集成 | 附加和整理旅行照片 |
| 地图视图 | 在交互式地图上可视化旅行路线 |
| 统计数据 | 追踪旅行指标和成就 |
| 分享 | 与他人分享冒险经历 |
| 导出 | 以多种格式导出数据 |
| 分类 | 按类型组织旅行 |
| 心愿清单 | 保存想去的地方 |
| 预算追踪 | 监控旅行开支 |

### 平台概览

```
AdventureLog Structure
├── Trips
│   ├── Basic Information
│   ├── Locations
│   ├── Dates
│   ├── Photos
│   ├── Notes
│   └── Expenses
├── Wishlist
│   ├── Destinations
│   ├── Notes
│   └── Priority
├── Statistics
│   ├── Countries Visited
│   ├── Distance Traveled
│   ├── Trip Duration
│   └── Expenses
└── Settings
    ├── Profile
    ├── Preferences
    └── Privacy
```

## 旅行日志

### 创建新旅行

要记录一次新旅行，请提供以下信息：

| 字段 | 必填 | 描述 |
|------|------|------|
| 标题 | 是 | 旅行名称 |
| 目的地 | 是 | 到访的地点 |
| 开始日期 | 是 | 旅行开始时间 |
| 结束日期 | 是 | 旅行结束时间 |
| 描述 | 否 | 旅行概要 |
| 分类 | 否 | 旅行类型 |
| 评分 | 否 | 你的评分（1-5 星） |
| 封面照片 | 否 | 旅行的主照片 |

### 旅行详情

为每次旅行添加详细信息：

**地点：**

| 字段 | 描述 |
|------|------|
| 国家 | 到访的国家 |
| 城市/地区 | 具体区域 |
| 地址 | 具体地址 |
| 坐标 | GPS 坐标 |
| 地点名称 | 具体地点名称 |

**活动：**

| 字段 | 描述 |
|------|------|
| 活动名称 | 你做了什么 |
| 日期 | 何时进行的 |
| 时长 | 花了多长时间 |
| 费用 | 花了多少钱 |
| 评分 | 你的评分 |
| 备注 | 补充详情 |

### 旅行时间线

按时间顺序组织你的旅行：

```
Trip: "Summer in Italy"
Duration: July 1-15, 2024

Day 1 (July 1): Arrival in Rome
├── Flight: NYC → Rome
├── Hotel check-in
└── Evening walk in Trastevere

Day 2 (July 2): Rome Sightseeing
├── Colosseum (morning)
├── Roman Forum (afternoon)
└── Dinner in Testaccio

Day 3 (July 3): Vatican City
├── Vatican Museums
├── Sistine Chapel
└── St. Peter's Basilica

...

Day 14 (July 14): Florence
├── Uffizi Gallery
├── Ponte Vecchio
└── Farewell dinner

Day 15 (July 15): Departure
├── Hotel checkout
└── Flight: Rome → NYC
```

| 条目类型 | 内容 |
|----------|------|

| 尺寸 | 图片大小 |

| 功能 | 描述 |

### 照片存储

| 存储选项 | 优点 | 缺点 |
|----------|------|------|
| 本地存储 | 速度快，无需网络 | 空间有限 |
| 云存储 | 随时随地访问 | 需要网络 |
| 外部硬盘 | 容量大 | 不一定随时可用 |
| NAS | 家庭网络访问 | 需要配置 |

### 照片备份策略

| 备份 | 位置 | 频率 |
|------|------|------|

| 副备份 | 外部硬盘 | 每周 |
| 第三备份 | 另一个云服务 | 每月 |

## 总结

AdventureLog 提供了一个完整的平台来记录和整理旅行体验：

- **旅行日志**：记录冒险的每个细节
- **照片集成**：附加和整理旅行回忆
- **地图视图**：在地图上可视化你的旅行
- **统计数据**：追踪旅行指标和成就
- **分享**：与他人分享体验
- **导出**：备份和导出你的数据
