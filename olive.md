## 简介

Olive 是一款免费、开源的非线性视频编辑器，旨在提供专业级编辑能力。它为视频创作者提供现代化界面和强大功能。

### Olive 核心功能

| 功能 | 描述 | 好处 |
|------|------|------|
| 非线性编辑 | 灵活的时间线 | 专业工作流 |
| 节点特效 | 视觉特效合成 | 高级特效 |
| GPU 加速 | 硬件渲染 | 快速性能 |
| 跨平台 | Windows, macOS, Linux | 广泛兼容 |
| 免费 | 无费用 | 人人可用 |

### Olive vs 其他编辑器

| 功能 | Olive | DaVinci Resolve | Kdenlive |
|------|-------|-----------------|----------|
| 价格 | 免费 | 免费/付费 | 免费 |
| 节点特效 | 是 | 是 | 否 |
| GPU 加速 | 是 | 是 | 有限 |
| 专业 | 是 | 是 | 部分 |
| 学习曲线 | 中等 | 高 | 中等 |

### 系统要求

| 组件 | 最低 | 推荐 |
|------|------|------|
| CPU | Intel i5 / AMD Ryzen 5 | Intel i7 / AMD Ryzen 7 |
| 内存 | 8GB | 16GB+ |
| GPU | OpenGL 4.3 | NVIDIA GTX 10+ |
| 存储 | 1GB | SSD |
| 显示 | 1920x1080 | 2560x1440 |

## 安装

在你的系统上安装 Olive 视频编辑器。

### 下载选项

| 平台 | 来源 | 文件类型 |
|------|------|----------|
| Windows | olive-editor.com | .exe 安装程序 |
| macOS | olive-editor.com | .dmg 文件 |
| Linux | 包管理器 | Flatpak, AppImage |

### 安装步骤

| 步骤 | 操作 | 备注 |
|------|------|------|
| 1 | 下载 | 官方网站 |
| 2 | 运行安装程序 | 接受提示 |
| 3 | 选择位置 | 默认路径 |
| 4 | 完成 | 启动 Olive |

### Linux 安装

```bash
# Flatpak
flatpak install flathub org.olivevideoeditor.Olive

# AppImage
chmod +x olive-*.AppImage
./olive-*.AppImage
```

## 导出

渲染最终视频输出。

### 导出设置

| 设置 | 选项 | 推荐 |
|------|------|------|
| 格式 | MP4, MOV, AVI | MP4 |
| 编码 | H.264, H.265 | H.264 |
| 分辨率 | 匹配源 | 1920x1080 |
| 帧率 | 匹配源 | 30/60 fps |
| 质量 | 可变 | High |

### 编码对比

| 编码 | 质量 | 文件大小 | 兼容性 |
|------|------|----------|--------|
| H.264 | 良好 | 中等 | 通用 |
| H.265 | 更好 | 更小 | 增长中 |
| ProRes | 优秀 | 大 | 专业 |
| DNxHR | 优秀 | 大 | 专业 |

### 导出流程

| 步骤 | 操作 | 备注 |
|------|------|------|
| 1 | 设置入/出点 | 选择范围 |
| 2 | File → Export | 打开对话框 |
| 3 | 选择设置 | 格式、编码 |
| 4 | 设置目标 | 输出文件夹 |
| 5 | 开始导出 | 渲染开始 |

### 导出预设

| 预设 | 分辨率 | 比特率 | 用途 |
|------|--------|--------|------|
| YouTube 1080p | 1920x1080 | 8 Mbps | 网页上传 |
| Vimeo 1080p | 1920x1080 | 10 Mbps | 网页上传 |
| Master | 原始 | 无损 | 归档 |
| Draft | 1280x720 | 2 Mbps | 预览 |

### 质量设置

| 质量 | CRF 值 | 文件大小 | 视觉质量 |
|------|--------|----------|----------|
| High | 18-22 | 较大 | 优秀 |
| Medium | 23-27 | 中等 | 良好 |
| Low | 28-32 | 较小 | 可接受 |
