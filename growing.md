## 简介

OpenFarm 是一个免费、开源的农业和园艺知识数据库。它为分享和发现任何作物或植物的种植指南提供平台。

### OpenFarm 核心功能

| 功能 | 描述 | 好处 |
|------|------|------|
| 开放数据 | 社区驱动内容 | 集体知识 |
| 结构化指南 | 有组织的种植信息 | 易于遵循 |
| API 访问 | 开发者友好 | 集成就绪 |
| 多语言 | 全球社区 | 全球可用 |
| 免费 | 无费用 | 向所有人开放 |

### OpenFarm vs 其他资源

| 功能 | OpenFarm | Garden.org | Extension Services |
|------|----------|------------|-------------------|
| 开源 | 是 | 否 | 否 |
| 结构化数据 | 是 | 部分 | 部分 |
| API | 是 | 否 | 否 |
| 社区 | 全球 | 区域 | 区域 |
| 成本 | 免费 | 免费 | 免费 |

### 数据类别

| 类别 | 内容 | 示例 |
|------|------|------|
| 作物 | 植物信息 | 番茄、玉米 |
| 指南 | 种植说明 | 分步指南 |
| 环境 | 生长条件 | 气候、土壤 |
| 实践 | 种植方法 | 有机、水培 |

## 生长阶段

定义植物的生命周期阶段。

### 常见生长阶段

| 阶段 | 描述 | 持续时间 |
|------|------|----------|
| 发芽 | 种子到幼苗 | 5-14 天 |
| 幼苗 | 早期生长 | 2-4 周 |
| 营养生长期 | 叶片发育 | 4-8 周 |
| 开花 | 花期 | 2-4 周 |
| 结果 | 果实发育 | 4-12 周 |
| 收获 | 可采摘 | 不定 |

### 阶段详情

| 字段 | 描述 | 示例 |
|------|------|------|
| Name | 阶段标题 | "发芽" |
| Description | 发生什么 | "种子发芽" |
| Duration | 时间跨度 | "7-14 天" |
| Actions | 做什么 | "保持湿润" |

### 添加阶段到指南

| 步骤 | 操作 | 备注 |
|------|------|------|
| 1 | 打开指南编辑器 | 编辑模式 |
| 2 | 点击"Add Stage" | 新阶段表单 |
| 3 | 输入阶段信息 | 名称、描述 |
| 4 | 设置持续时间 | 天/周 |
| 5 | 添加操作 | 说明 |
| 6 | 保存 | 阶段创建完成 |

### 阶段特定操作

| 阶段 | 常见操作 |
|------|----------|
| 发芽 | 保持湿润、温暖温度 |
| 幼苗 | 移栽、间苗 |
| 营养生长期 | 施肥、修剪 |
| 开花 | 授粉、支撑 |
| 结果 | 持续浇水 |
| 收获 | 成熟时采摘 |

## 环境要求

定义最佳生长条件。

### 环境因素

| 因素 | 描述 | 范围 |
|------|------|------|
| 阳光 | 光照 | 全日照、半荫、荫 |
| 水 | 灌溉需求 | 低、中、高 |
| 土壤 | 土壤类型 | 沙质、壤土、粘土 |
| 温度 | 生长温度 | 最低、最佳、最高 |
| pH | 土壤酸碱度 | 0-14 标度 |

### 阳光要求

| 级别 | 时数 | 示例 |
|------|------|------|
| 全日照 | 6+ 小时 | 番茄、辣椒 |
| 半日照 | 4-6 小时 | 生菜、菠菜 |
| 半荫 | 2-4 小时 | 蕨类、玉簪 |
| 全荫 | < 2 小时 | 蘑菇 |

### 水分要求

| 级别 | 频率 | 示例 |
|------|------|------|
| 低 | 每周 | 仙人掌、多肉 |
| 中 | 2-3 次/周 | 大多数蔬菜 |
| 高 | 每天 | 水稻、西洋菜 |

### 土壤类型

| 类型 | 排水 | 养分 | 最适合 |
|------|------|------|--------|
| 沙质 | 快 | 低 | 根茎类 |
| 壤土 | 中 | 高 | 大多数作物 |
| 粘土 | 慢 | 高 | 重肥作物 |
| 粉土 | 中 | 中 | 喜湿作物 |

### 气候带

| 带 | 温度 | 示例 |
|----|------|------|
| 热带 | 全年 > 64°F | 香蕉、芒果 |
| 亚热带 | 冬季 50-64°F | 柑橘、牛油果 |
| 温带 | 冬季 32-50°F | 苹果、小麦 |
| 大陆性 | 冬季 < 32°F | 耐寒作物 |
| 极地 | 全年 < 32°F | 有限作物 |

## API 与集成

开发者使用 OpenFarm API。

### API 端点

| 端点 | 方法 | 描述 |
|------|------|------|
| /api/v1/crops | GET | 列出作物 |
| /api/v1/crops/:id | GET | 获取作物 |
| /api/v1/guides | GET | 列出指南 |
| /api/v1/guides/:id | GET | 获取指南 |

### API 认证

| 方式 | Header | 用途 |
|------|--------|------|
| Token | Authorization | 认证请求 |
| 无 | - | 公开数据 |

### API 示例

```bash
# Get all crops
curl https://api.openfarm.cc/api/v1/crops

# Get specific crop
curl https://api.openfarm.cc/api/v1/crops/1

# Search crops
curl https://api.openfarm.cc/api/v1/crops?filter=tomato
```

### 集成创意

| 集成 | 描述 | 用途 |
|------|------|------|
| 花园规划器 | 可视布局 | 规划 |

| 移动应用 | 现场参考 | 随时随地 |
| 日历 | 种植计划 | 时间安排 |

### API 速率限制

| 层级 | 请求/分钟 | 请求/天 |
