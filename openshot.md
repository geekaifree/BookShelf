## 简介

OpenShot 是一款免费、开源的视频编辑器，提供易于使用的界面和强大功能。适合初学者和中级视频编辑者。

### OpenShot 核心功能

| 功能 | 描述 | 好处 |
|------|------|------|
| 跨平台 | Windows, macOS, Linux | 广泛兼容 |
| 免费 | 无费用 | 人人可用 |
| 简洁界面 | 拖放 | 快速学习 |
| 无限轨道 | 多图层 | 复杂项目 |
| 3D 动画 | 标题特效 | 专业外观 |

### OpenShot vs 其他编辑器

| 功能 | OpenShot | Shotcut | Kdenlive |
|------|----------|---------|----------|
| 价格 | 免费 | 免费 | 免费 |
| 易用性 | 简单 | 中等 | 中等 |
| 功能 | 良好 | 良好 | 高级 |
| 性能 | 良好 | 良好 | 良好 |
| 稳定性 | 良好 | 良好 | 良好 |

### 系统要求

| 组件 | 最低 | 推荐 |
|------|------|------|
| CPU | 多核处理器 | 现代多核 |
| 内存 | 4GB | 8GB+ |
| GPU | OpenGL 2.1 | OpenGL 3.3+ |
| 存储 | 500MB | SSD |
| 显示 | 1024x768 | 1920x1080 |

## 安装

安装 OpenShot 视频编辑器。

### 下载选项

| 平台 | 来源 | 文件类型 |
|------|------|----------|
| Windows | openshot.org | .exe 安装程序 |
| macOS | openshot.org | .dmg 文件 |
| Linux | 包管理器 | apt, flatpak, AppImage |

### 安装步骤

| 步骤 | 操作 | 备注 |
|------|------|------|
| 1 | 下载 | 官方网站 |
| 2 | 运行安装程序 | 接受提示 |
| 3 | 选择位置 | 默认路径 |
| 4 | 完成 | 启动 OpenShot |

### Linux 安装

```bash
# Ubuntu/Debian
sudo apt install openshot-qt

# Flatpak
flatpak install flathub org.openshot.OpenShot

# AppImage
chmod +x openshot-*.AppImage
./openshot-*.AppImage
```

## 导出

渲染最终视频输出。

### 导出设置

| 设置 | 选项 | 推荐 |
|------|------|------|
| 格式 | MP4, AVI, MOV | MP4 |
| 配置 | Web, HD 等 | 根据需要 |
| 分辨率 | 720p, 1080p | 1080p |
| 帧率 | 24, 30, 60 | 30 fps |
| 质量 | Low 到 High | High |

### 导出预设

| 预设 | 分辨率 | 比特率 | 用途 |
|------|--------|--------|------|
| Web | 1280x720 | 5 Mbps | 在线分享 |
| HD | 1920x1080 | 8 Mbps | 高质量 |
| Full HD | 1920x1080 | 15 Mbps | 最佳质量 |
| 4K | 3840x2160 | 30 Mbps | 超高清 |

### 导出流程

| 步骤 | 操作 | 备注 |
|------|------|------|
| 1 | 设置入/出 | 选择范围 |
| 2 | File → Export | 导出对话框 |
| 3 | 选择设置 | 格式、质量 |
| 4 | 设置目标 | 输出文件夹 |
| 5 | 开始导出 | 渲染开始 |

### 导出格式对比

| 格式 | 质量 | 文件大小 | 兼容性 |
|------|------|----------|--------|
| MP4 | 良好 | 中等 | 通用 |
| AVI | 良好 | 大 | Windows |
| MOV | 良好 | 中等 | macOS |
| WebM | 良好 | 小 | Web |

### 质量设置

| 质量 | CRF | 文件大小 | 视觉质量 |
|------|-----|----------|----------|
| Low | 28-32 | 小 | 可接受 |
| Medium | 23-27 | 中等 | 良好 |
| High | 18-22 | 大 | 优秀 |
| Very High | 0-17 | 非常大 | 无损 |
