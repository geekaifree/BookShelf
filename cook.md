## 烹饪技巧

### 炒（炒）

炒是最常见的中餐烹饪技巧，使用大火快速翻炒。

**步骤：**

1. 将炒锅烧至冒烟
2. 加油并转动锅底使其均匀覆盖
3. 加入香料（蒜、姜）爆香 10 秒
4. 加入蛋白质食材，炒至七成熟
5. 盛出蛋白质食材备用
6. 加入蔬菜翻炒
7. 将蛋白质食材倒回锅中
8. 加入酱汁，翻炒均匀
9. 立即出锅

**关键要点：**

- 食材要不断翻动
- 食材切至均匀大小以确保受热均匀
- 开火前准备好所有食材（mise en place）

### 炸（炸）

| 温度 | 效果 | 示例 |
|-------------|--------|----------|
| 150-160°C (300-320°F) | 温和烹饪 | 蔬菜、精致食材 |
| 170-180°C (340-360°F) | 标准油炸 | 春卷、鸡肉 |
| 190-200°C (375-400°F) | 快速脆化 | 二次炸制增加酥脆度 |

**复炸技巧：**

1. 第一次低温炸至熟透
2. 静置 5-10 分钟
3. 第二次高温炸至酥脆

### 蒸（蒸）

蒸制能保留营养和天然风味。

| 食材 | 蒸制时间 |
|------|---------------|
| 鱼片 | 8-10 分钟 |
| 整鱼（1 磅） | 12-15 分钟 |
| 饺子 | 10-12 分钟 |
| 包子 | 15-18 分钟 |
| 蔬菜 | 5-8 分钟 |

**技巧：**

- 水烧开后再放入食材
- 蒸制过程中不要掀盖
- 食材之间留出空间以便蒸汽流通

### 红烧（红烧）

红烧能做出浓郁可口、肉质软烂的菜肴。

**基本步骤：**

1. 热油将肉煎至金黄
2. 加入香料和调味料
3. 加入液体（高汤、酱油、料酒）
4. 小火慢炖 1-2 小时
5. 收汁至浓稠

| 肉类 | 红烧时间 |
|------|---------------|
| 鸡块 | 30-40 分钟 |
| 五花肉 | 1.5-2 小时 |
| 牛腩 | 2-3 小时 |
| 羊肉 | 2-2.5 小时 |

### 焯水（焯水）

焯水可以去除杂质并预处理食材。

**目的：**

- 去除肉类的血水和杂质
- 固定绿色蔬菜的颜色
- 缩短致密蔬菜的烹饪时间
- 去除某些食材的苦味

**方法：**

1. 大锅烧开水
2. 加入食材
3. 短暂焯烫（30 秒至 2 分钟）
4. 捞出后放入冰水中（蔬菜用）
5. 沥干后按需使用

## 食品安全

### 温度指南

| 食材类型 | 安全内部温度 |
|-----------|---------------------------|
| 鸡肉 | 75°C (165°F) |
| 猪肉 | 63°C (145°F) |
| 牛肉（五分熟） | 63°C (145°F) |
| 鱼 | 63°C (145°F) |
| 鸡蛋 | 71°C (160°F) |

### 防止交叉污染

| 做法 | 说明 |
|----------|-------------|
| 分开砧板 | 生肉和蔬菜使用不同的砧板 |
| 洗手 | 处理生肉前后都要洗手 |
| 清洁台面 | 食材处理后消毒台面 |
| 存放生肉 | 始终放在即食食品下方 |
| 分开器具 | 生熟食品不使用同一工具 |

### 危险温度区间

细菌在 4°C 至 60°C (40°F-140°F) 之间快速繁殖。

- 食物在室温下放置不要超过 2 小时
- 热食冷却后再放入冰箱
- 剩菜加热至 74°C (165°F)

## 分步菜谱

### 番茄炒蛋（番茄炒蛋）

**食材：**

| 食材 | 用量 |
|------------|--------|
| 鸡蛋 | 4 个（大） |
| 番茄 | 3 个（中） |
| 糖 | 1 茶匙 |
| 盐 | 1/2 茶匙 |
| 葱 | 2 根 |
| 油 | 3 汤匙 |

**步骤：**

1. 鸡蛋加少许盐打散
2. 番茄切块
3. 锅中加入 2 汤匙油烧热
4. 倒入鸡蛋，炒至刚凝固，盛出
5. 加 1 汤匙油
6. 加入番茄，炒至变软
7. 加糖和盐
8. 将鸡蛋倒回锅中
9. 翻炒均匀，撒葱花

### 宫保鸡丁（宫保鸡丁）

**食材：**

| 食材 | 用量 |
|------------|--------|
| 鸡胸肉 | 300g |
| 干辣椒 | 8-10 个 |
| 花椒 | 1 汤匙 |
| 花生 | 1/3 杯 |
| 葱 | 2 根 |
| 姜 | 3 片 |
| 蒜 | 3 瓣 |

**腌料：**

| 食材 | 用量 |
|------------|--------|
| 生抽 | 1 汤匙 |
| 料酒 | 1 汤匙 |
| 淀粉 | 1 茶匙 |

**酱汁：**

| 食材 | 用量 |
|------------|--------|
| 生抽 | 1 汤匙 |
| 老抽 | 1/2 汤匙 |
| 黑醋 | 1 汤匙 |
| 糖 | 1 汤匙 |
| 淀粉 | 1 茶匙 |
| 水 | 2 汤匙 |

**步骤：**

1. 鸡肉切 1 英寸丁
2. 加入腌料拌匀，腌制 15 分钟
3. 将酱汁材料在碗中混合
4. 锅烧至冒烟
5. 加油，放入干辣椒和花椒（10 秒）
6. 加入鸡肉，炒至金黄
7. 加入姜、蒜、葱
8. 倒入酱汁，炒至浓稠
9. 加入花生，翻炒均匀
10. 立即出锅

### 麻婆豆腐（麻婆豆腐）

**食材：**

| 食材 | 用量 |
|------------|--------|
| 嫩豆腐 | 1 块（400g） |
| 猪肉末 | 100g |
| 豆瓣酱 | 2 汤匙 |
| 豆豉 | 1 汤匙 |
| 蒜 | 3 瓣 |
| 姜 | 2 片 |
| 葱 | 2 根 |
| 花椒粉 | 1 茶匙 |

**步骤：**

1. 豆腐切 1 英寸方块，盐水焯烫
2. 热油炒肉末至金黄
3. 加入豆瓣酱，炒出红油
4. 加入蒜、姜、豆豉
5. 加 1 杯水烧开
6. 轻轻放入豆腐，小火炖 5 分钟
7. 用水淀粉勾芡
8. 撒葱花和花椒粉

### 炒饭（炒饭）

**关键原则：**

- 使用隔夜冷饭
- 大火是关键
- 动作要快
- 不要放太多食材

**食材：**

| 食材 | 用量 |
|------------|--------|
| 米饭（冷的） | 3 杯 |
| 鸡蛋 | 2 个 |
| 葱 | 3 根 |
| 盐 | 1/2 茶匙 |
| 白胡椒 | 1/4 茶匙 |
| 油 | 3 汤匙 |

**步骤：**

1. 用手将冷饭打散
2. 鸡蛋加少许盐打散
3. 锅烧至冒烟
4. 加油转动锅底
5. 倒入鸡蛋，炒至半凝固
6. 蛋液未完全凝固时加入米饭
7. 翻炒，打散结块
8. 加盐和白胡椒调味
9. 加入葱花，翻炒均匀
10. 立即出锅

## 常见错误

### 烹饪错误及修正

| 错误 | 问题 | 解决方案 |
|---------|---------|----------|
| 锅不够热 | 食材变成焖煮而非煎炒 | 烧至油冒烟 |
| 锅中食材过多 | 温度下降，食材焖煮 | 分批炒制 |
| 翻炒太多 | 不上色，口感软烂 | 让食材稍作停留 |
| 酱汁加太早 | 稀释味道，食材变软 | 最后加入 |
| 油温太低 | 粘锅，味道差 | 烧至油微冒烟 |
| 用错酱油 | 太咸或颜色不对 | 生抽调味，老抽上色 |
| 蔬菜炒太久 | 软烂、变色 | 大火快炒 |
| 没有提前备菜 | 手忙脚乱，食材过熟 | 开火前准备好一切 |

### 米饭问题

| 问题 | 原因 | 解决方案 |
|---------|-------|-----|
| 米饭太烂 | 水太多 | 淘洗后米水比 1:1 |
| 米饭太硬 | 水太少 | 多加 1-2 汤匙水 |
| 米饭糊了 | 火太大 | 用电饭煲或小火 |
| 米饭粘连 | 没淘洗 | 淘洗至水清 |

## 新手技巧

1. **从简单开始** - 先掌握基础菜肴再尝试复杂的
2. **提前备好所有食材** - 中餐大火炒制速度很快
3. **边做边尝** - 烹饪过程中随时调整调味
4. **使用大火** - 大多数炒菜需要最大火力
5. **保持翻动** - 炒菜需要不断翻炒
6. **不要怕油** - 适量的油能防止粘锅并增加风味
7. **学会镬气** - 镬气来自大火和不断练习
8. **使用新鲜食材** - 优质的食材做出更好的菜
9. **边做边收拾** - 中餐会产生很多餐具
10. **勤加练习** - 技能随重复而提升
