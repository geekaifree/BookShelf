## 🌟 项目简介

BookShelf 是一个 **多语言资源与教程集合**，汇集了各领域的精选学习资料、工具、最佳实践以及深度教程。每个文档都经过精心整理，以中文呈现，帮助用户快速获取高质量资源。

- 🌍 **多领域覆盖** — 技术、金融、设计、生活等 15+ 个领域
- 📚 **教程驱动** — 50 篇精编教程，从入门到进阶
- 📱 **移动友好** — Markdown 格式，随时随地阅读
- 🔍 **持续更新** — 定期补充最新资源与工具
- 📊 **表格展示** — 清晰易读，快速查找
