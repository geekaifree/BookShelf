# Spotube：开源音乐串流应用

## 简介

Spotube 是一款免费、开源的音乐串流客户端，可连接 Spotify。它无需 Spotify Premium 订阅即可访问数百万首歌曲。基于 Flutter 构建，Spotube 可在 Windows、macOS、Linux、Android 和 iOS 等多个平台上运行。

**核心功能：**

- 无需 Premium 即可免费串流音乐
- 无广告或追踪
- 支持离线播放
- 同步歌词显示
- 播放列表管理
- 跨平台兼容
- 注重隐私设计

## 安装

### 系统要求

| 平台 | 最低版本 | 存储空间 | 内存 |
|----------|----------------|---------|-----|
| Windows | Windows 10 | 200 MB | 2 GB |
| macOS | 11.0 Big Sur | 200 MB | 2 GB |
| Linux | Ubuntu 20.04+ | 200 MB | 2 GB |
| Android | Android 7.0 | 150 MB | 1 GB |
| iOS | iOS 14.0 | 150 MB | 1 GB |

### Windows 安装

从官方发布页面下载安装程序：

1. 访问 GitHub 发布页面
2. 下载 `.exe` 安装文件
3. 运行安装程序并按向导操作
4. 从开始菜单启动 Spotube

### macOS 安装

通过 Homebrew 安装或直接下载：

```bash
brew install --cask spotube
```

或者下载 `.dmg` 文件并拖入应用程序文件夹。

### Linux 安装

支持多种安装方式：

**Flatpak（推荐）：**
```bash
flatpak install flathub com.github.KRTirtho.Spotube
```

**Snap：**
```bash
sudo snap install spotube
```

**Debian/Ubuntu（.deb）：**
```bash
sudo dpkg -i spotube-linux-x86_64.deb
sudo apt-get install -f
```

**AppImage：**
```bash
chmod +x spotube-linux-x86_64.AppImage
./spotube-linux-x86_64.AppImage
```

### 移动端安装

| 平台 | 来源 | 步骤 |
|----------|--------|-------|
| Android | GitHub Releases | 下载 APK，启用未知来源，安装 |
| Android | F-Droid | 从 F-Droid 商店搜索并安装 |
| iOS | AltStore | 使用 AltStore 或类似工具侧载 |

## 初始设置

### Spotify 账户连接

Spotube 需要 Spotify 账户来访问音乐元数据。设置流程：

1. 启动 Spotube
2. 点击"连接 Spotify"
3. 使用 Spotify 凭据登录
4. 授权 Spotube 访问您的账户
5. 等待同步完成

### 音频源配置

Spotube 使用 YouTube 或 Piped 作为音频源。配置首选源：

| 源 | 质量 | 速度 | 可靠性 |
|--------|---------|-------|-------------|
| YouTube | 高 | 中等 | 非常高 |
| Piped | 高 | 快 | 高 |
| Invidious | 中等 | 快 | 中等 |

前往 设置 > 音频源 选择您的偏好。

## 功能

### 音乐搜索与播放

Spotube 提供全面的搜索界面：

- 按歌曲名称、艺术家或专辑搜索
- 浏览精选播放列表
- 访问您的 Spotify 资料库
- 查看最近播放的曲目
- 发现新音乐推荐

**播放控制：**

| 控制 | 功能 |
|---------|----------|
| 播放/暂停 | 开始或停止播放 |
| 下一首/上一首 | 在曲目间切换 |
| 进度条 | 跳转到指定位置 |
| 音量 | 调整音量 |
| 随机播放 | 随机排列曲目顺序 |
| 循环播放 | 循环播放单首或整个播放列表 |

### 歌词显示

Spotube 为支持的曲目显示同步歌词：

1. 播放任意歌曲
2. 点击播放栏中的歌词图标
3. 歌词随音乐自动滚动
4. 点击任意行跳转到该时间点

**歌词来源：**

| 提供商 | 覆盖率 | 同步质量 |
|----------|----------|--------------|
| Musixmatch | 95% | 优秀 |
| Genius | 85% | 良好（仅文本） |
| LRCLIB | 70% | 优秀 |

### 播放列表管理

在 Spotube 中直接创建和管理播放列表：

**创建播放列表：**
1. 进入库标签
2. 点击"创建播放列表"
3. 输入名称和可选描述
4. 添加封面图片（可选）
5. 开始添加曲目

**播放列表操作：**

| 操作 | 描述 |
|-----------|-------------|
| 添加曲目 | 搜索并添加单首歌曲 |
| 删除曲目 | 滑动或右键删除 |
| 重新排序 | 拖放重新排列曲目 |
| 分享 | 生成可分享的播放列表链接 |
| 导出 | 导出为 M3U 或其他格式 |
| 克隆 | 创建现有播放列表的副本 |

### 离线模式

下载歌曲供离线收听：

1. 找到想要的歌曲或播放列表
2. 点击下载图标
3. 选择音频质量
4. 等待下载完成
5. 在离线部分访问下载内容

**下载质量选项：**

| 质量 | 比特率 | 文件大小（每首） | 使用场景 |
|---------|---------|---------------------|----------|
| 低 | 96 kbps | 约 2 MB | 节省流量 |
| 中等 | 160 kbps | 约 4 MB | 日常使用 |
| 高 | 256 kbps | 约 6 MB | 好音质 |
| 极高 | 320 kbps | 约 8 MB | 最佳音质 |

## 故障排除

### 常见问题

| 问题 | 解决方案 |
|-------|----------|
| 无音频播放 | 检查音频源设置，尝试切换源 |
| 歌曲加载失败 | 清除缓存，重启应用 |
| 登录失败 | 重新认证 Spotify 连接 |
| 歌词不显示 | 尝试不同的歌词提供商 |
| 下载失败 | 检查存储空间和网络连接 |

### 性能优化

1. **清除缓存：** 设置 > 存储 > 清除缓存
2. **降低质量：** 为较慢连接降低串流质量
3. **禁用动画：** 设置 > 辅助功能 > 减少动画
4. **定期更新：** 保持 Spotube 更新到最新版本
