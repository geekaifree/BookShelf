## 浏览器

### 为什么浏览器隐私很重要

浏览器是你访问互联网的主要界面。它收集大量浏览习惯数据，这些数据可用于追踪和画像。

### 注重隐私的浏览器

| 浏览器 | 平台 | 开源 | 隐私功能 |
|---------|----------|-------------|------------------|
| Firefox | 全平台 | 是 | 增强追踪保护、容器 |
| Brave | 全平台 | 是 | 内置广告拦截、Tor 集成 |
| Tor Browser | 全平台 | 是 | 最大匿名性、洋葱路由 |
| LibreWolf | 桌面 | 是 | 加固版 Firefox、无遥测 |
| Mull | Android | 是 | 注重隐私的 Firefox 分支 |
| Bromite | Android | 是 | 带隐私补丁的 Chromium |
| Safari | Apple | 否 | 智能防追踪 |

### 浏览器对比

| 功能 | Firefox | Brave | Tor Browser |
|---------|---------|-------|-------------|
| 速度 | 快 | 快 | 慢 |
| 隐私 | 高 | 高 | 最高 |
| 兼容性 | 优秀 | 优秀 | 良好 |
| 扩展 | 丰富 | 丰富 | 有限 |
| 指纹防护 | 好 | 好 | 优秀 |
| 内置广告拦截 | 否 | 是 | 否 |
| Tor 集成 | 否 | 是 | 内置 |

### 推荐浏览器扩展

| 扩展 | 用途 | 平台 |
|-----------|---------|----------|
| uBlock Origin | 广告和追踪器拦截 | 全平台 |
| Privacy Badger | 追踪器拦截 | 全平台 |
| Cookie AutoDelete | 自动删除 Cookie | Firefox、Chrome |
| NoScript | JavaScript 控制 | Firefox |
| HTTPS Everywhere | 强制 HTTPS | 全平台 |
| Bitwarden | 密码管理器 | 全平台 |
| ClearURLs | 移除追踪参数 | Firefox、Chrome |

### 浏览器配置建议

| 设置 | 推荐 |
|---------|----------------|
| 追踪保护 | 启用严格模式 |
| 第三方 Cookie | 全部阻止 |
| HTTPS-only 模式 | 启用 |
| 遥测 | 禁用 |
| 自动填充 | 密码禁用 |
| 隐私浏览 | 敏感网站使用 |
| DNS over HTTPS | 使用隐私 DNS 启用 |
