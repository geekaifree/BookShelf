## 了解原料

### 面粉

面粉是面包的基础。不同面粉蛋白质含量不同，影响面筋的形成。

| 面粉类型 | 蛋白质含量 | 最佳用途 |
|------------|-----------------|----------|
| 高筋面粉 | 12-14% | 乡村面包、披萨 |
| 中筋面粉 | 10-12% | 吐司、小餐包 |
| 全麦面粉 | 13-14% | 密实、营养面包 |
| 黑麦粉 | 8-10% | 黑麦面包、粗麦面包 |
| 斯佩尔特面粉 | 10-12% | 坚果风味面包 |
| 单粒小麦粉 | 9-11% | 古老谷物面包 |

**蛋白质与面筋：**

- 蛋白质越高 = 面筋越多 = 口感越有嚼劲
- 蛋白质越低 = 面筋越少 = 口感越松软
- 面筋赋予面包结构和嚼劲

### 水

| 因素 | 影响 |
|--------|--------|
| 温度 | 影响发酵速度 |
| 矿物质含量 | 硬水能增强面筋 |
| 用量（水合度） | 决定面包内部组织 |

**水温指南：**

| 目标面团温度 | 夏季水温 | 冬季水温 |
|--------------------|---------------------|---------------------|
| 24°C (75°F) | 冷自来水 | 室温 |
| 26°C (79°F) | 室温 | 微温 |
| 28°C (82°F) | 微温 | 温热 |

### 酵母

| 类型 | 说明 | 用法 |
|------|-------------|-------|
| 活性干酵母 | 颗粒状，需要活化 | 先用温水溶解 |
| 即发酵母 | 细颗粒，无需活化 | 直接与面粉混合 |
| 鲜酵母 | 块状，易变质 | 掉入面团中 |
| 天然酵种 | 自然发酵 | 需要定期喂养 |

**酵母换算：**

| 鲜酵母 | 活性干酵母 | 即发酵母 |
|-------------|------------------|---------------|
| 10g | 5g | 4g |
| 30g | 15g | 12g |

### 盐

| 类型 | 特点 | 用途 |
|------|-----------------|-------|
| 细盐 | 溶解快 | 面包标准用盐 |
| 海盐 | 矿物质风味 | 特色面包 |
| 粗盐 | 颗粒大 | 需调整用量 |

**盐的功能：**

- 增强面筋结构
- 控制发酵速度
- 提升风味
- 延长保质期

**推荐用量：** 面粉重量的 2%

### 其他原料

| 原料 | 用途 | 典型用量 |
|------------|---------|----------------|
| 糖 | 喂养酵母、上色 | 0-5% |
| 油脂（黄油、油） | 松软、保质期 | 0-10% |
| 鸡蛋 | 丰富口感、上色 | 0-15% |
| 牛奶 | 松软、风味 | 0-50% |
| 种子/坚果 | 风味、口感 | 5-20% |

## 烘焙百分比

烘焙百分比以面粉重量为基准表示各原料的比例，便于等比缩放。

### 基本公式

    百分比 = (原料重量 / 面粉重量) x 100

### 示例配方（70% 水合度）

| 原料 | 重量 | 烘焙百分比 |
|------------|--------|-------------------|
| 高筋面粉 | 500g | 100% |
| 水 | 350g | 70% |
| 盐 | 10g | 2% |
| 即发酵母 | 5g | 1% |
| **合计** | **865g** | **173%** |

### 常见水合度

| 水合度 | 内部组织 | 示例 |
|-----------|------------|----------|
| 55-60% | 紧密、均匀 | 吐司、贝果 |
| 65-70% | 开放、松软 | 乡村面包、佛卡夏 |
| 75-80% | 非常开放、蓬松 | 恰巴塔、乡村面包 |
| 80%+ | 极度开放 | 佛卡夏、部分酸面团 |

### 缩放配方

做 2 个而不是 1 个面包：

1. 所有重量乘以 2
2. 百分比保持不变
3. 必要时调整发酵时间

## 面包制作流程

### 总览

| 阶段 | 时长 | 目的 |
|-------|----------|---------|
| 混合 | 10-15 分钟 | 混合原料 |
| 水合 | 20-60 分钟 | 面粉吸水、面筋形成 |
| 揉面 | 10-20 分钟 | 发展面筋结构 |
| 一次发酵 | 2-6 小时 | 风味发展、膨胀 |
| 整形 | 5-10 分钟 | 建立结构 |
| 最终发酵 | 1-2 小时 | 最后膨胀 |
| 烘烤 | 30-45 分钟 | 固定结构、形成外壳 |

### 分步流程

**1. 混合**

1. 称量所有原料
2. 将面粉和水混合（预留部分水）
3. 搅拌至无干粉
4. 静置 20-60 分钟（水合）
5. 加入盐和酵母
6. 搅拌均匀

**2. 一次发酵**

- 面团在室温下发酵
- 每 30 分钟做一次拉伸折叠
- 总时间：2-6 小时，取决于温度
- 面团应膨胀 50-75%

**3. 整形**

- 将面团倒在撒了少许面粉的台面上
- 预整形为圆形或长方形
- 静置 15-20 分钟
- 最终整形（圆形、橄榄形、法棍）

**4. 最终发酵**

- 将整形好的面团放入发酵篮或烤盘
- 盖好发酵至蓬松
- 指压测试：面团应缓慢回弹
- 时间：室温下 1-2 小时

**5. 烘烤**

- 将铸铁锅放入烤箱预热
- 用刀片在面团上割包
- 盖盖烤 20 分钟
- 揭盖再烤 15-20 分钟
- 内部温度：95-99°C (203-210°F)

## 揉面技巧

### 手揉面团

**拉伸折叠法：**

1. 手沾水防粘
2. 抓住面团一侧
3. 向上拉伸
4. 折回中心
5. 旋转 90 度
6. 重复 5-10 分钟

**摔打折叠法：**

1. 台面沾湿
2. 将面团摔在台面上
3. 向自己方向拉伸
4. 折回覆盖
5. 旋转并重复

### 厨师机

| 速度 | 时间 | 目的 |
|-------|------|---------|
| 低速（1-2） | 2 分钟 | 初步混合 |
| 中速（3-4） | 8-10 分钟 | 面筋发展 |
| 高速（5-6） | 2-3 分钟 | 最终发展 |

**警告：** 不要过度揉面。面团应光滑有弹性，而非硬实。

### 窗膜测试

检查面团是否揉好：

1. 取一小块面团
2. 用手指轻轻拉伸
3. 如果能拉到透光而不破，面筋已形成
4. 如果轻易撕裂，需要继续揉

## 发酵

### 一次发酵

| 温度 | 时长 | 效果 |
|-------------|----------|--------|
| 18°C (64°F) | 6-8 小时 | 缓慢、风味复杂 |
| 22°C (72°F) | 4-5 小时 | 风味均衡 |
| 25°C (77°F) | 3-4 小时 | 较快、风味较淡 |
| 28°C (82°F) | 2-3 小时 | 快速、风味简单 |

### 拉伸折叠

在一次发酵期间进行，无需揉面即可发展面筋。

**时间安排：**

| 时间 | 动作 |
|------|--------|
| 0 分钟 | 开始一次发酵 |
| 30 分钟 | 第一次拉伸折叠 |
| 60 分钟 | 第二次 |
| 90 分钟 | 第三次 |
| 120 分钟 | 第四次（可选） |

### 发酵正常的标志

- 面团膨胀 50-75%
- 表面出现小气泡
- 面团手感轻盈、有弹性
- 指压测试：凹痕缓慢回弹

### 过度发酵的标志

- 面团有浓烈酒精味
- 非常松散粘手
- 表面出现大气泡
- 指压后塌陷

## 整形

### 圆形面包（Boule）

1. 预整形为圆形，静置 15-20 分钟
2. 翻面（光滑面朝下）
3. 边缘向中心折叠
4. 再翻面（接缝朝下）
5. 双手环绕面团
6. 向自己方向拖拽制造张力
7. 接缝朝上放入发酵篮

### 橄榄形面包（Batard）

1. 预整形为椭圆形，静置 15-20 分钟
2. 翻面（光滑面朝下）
3. 上三分之一向下折叠，按压接缝
4. 下三分之一向上折叠，按压接缝
5. 对折，封好接缝
6. 接缝朝上放入发酵篮

### 法棍（Baguette）

1. 预整形为长方形，静置 15-20 分钟
2. 上三分之一向下折叠，按压接缝
3. 下三分之一向上折叠，按压接缝
4. 卷成长圆柱
5. 接缝朝下放在帆布上

### 制造表面张力

- 最终整形使用无面粉的台面
- 用刮刀将面团在台面上拖拽
- 面团应感觉紧绷，像气球一样
- 张力不足 = 扁平面包

## 烘烤

### 烤箱设置

| 方法 | 设置 | 效果 |
|--------|-------|--------|
| 铸铁锅 | 带盖预热，先盖后揭烤 | 最佳外壳，最简单 |
| 烤石 | 预热石头，加蒸汽 | 好外壳，底部酥脆 |
| 烤盘 | 预热烤盘，加蒸汽 | 尚可外壳，操作简单 |

### 蒸汽

蒸汽在烘烤初期保持外壳柔软，使面包充分膨胀。

**制造蒸汽的方法：**

| 方法 | 操作 |
|--------|--------|
| 铸铁锅 | 锅盖锁住面团产生的蒸汽 |
| 开水 | 倒入烤箱底部预热的烤盘 |
| 喷壶 | 喷烤箱壁和面团 |
| 冰块 | 放在预热烤盘上 |

### 割包

割包控制面包在烘烤时的膨胀方向。

| 图案 | 用途 |
|---------|---------|
| 单刀 | 传统，允许膨胀 |
| 十字 | 装饰性，均匀膨胀 |
| 麦穗图案 | 装饰性，质朴外观 |
| 方形 | 圆形面包用 |

**割包技巧：**

- 使用锋利的刀片或割刀
- 割 1/4 到 1/2 英寸深
- 刀片角度 30 度
- 快速、果断的动作
- 烘烤前即刻割包

### 烘烤时间和温度

| 面包类型 | 温度 | 时间 |
|------------|-------------|------|
| 酸面团圆包 | 240°C (475°F) | 40-45 分钟 |
| 法棍 | 240°C (475°F) | 20-25 分钟 |
| 吐司 | 190°C (375°F) | 30-35 分钟 |
| 佛卡夏 | 220°C (425°F) | 20-25 分钟 |
| 小餐包 | 190°C (375°F) | 15-20 分钟 |

### 烘熟标志

| 标志 | 说明 |
|------|-------------|
| 颜色 | 深金棕色 |
| 声音 | 敲底部发出中空声 |
| 温度 | 内部温度 95-99°C (203-210°F) |
| 重量 | 拿起来感觉轻 |

## 天然酵种

### 什么是天然酵种？

天然酵种是面粉和水的混合物，从环境中捕获野生酵母和细菌。它替代商业酵母并增添复杂风味。

### 制作酵种

**每日指南：**

| 天数 | 操作 | 备注 |
|-----|--------|-------|
| 第 1 天 | 50g 面粉 + 50g 水混合 | 用全麦或黑麦粉 |
| 第 2 天 | 弃一半，加 50g 面粉 + 50g 水 | 可能看到一些活动 |
| 第 3 天 | 弃一半，加 50g 面粉 + 50g 水 | 应该有气泡 |
| 第 4 天 | 弃一半，加 50g 面粉 + 50g 水 | 起起落落 |
| 第 5 天 | 弃一半，加 50g 面粉 + 50g 水 | 应在 4-6 小时内翻倍 |
| 第 6 天 | 弃一半，加 50g 面粉 + 50g 水 | 活动稳定 |
| 第 7 天 | 可以使用 | 4-6 小时内翻倍 |

### 维护酵种

**室温（每日喂养）：**

| 操作 | 用量 |
|--------|--------|
| 弃掉 | 只留 50g |
| 加面粉 | 50g |
| 加水 | 50g |

**冰箱（每周喂养）：**

| 操作 | 用量 |
|--------|--------|
| 从冰箱取出 | 回温 1-2 小时 |
| 弃掉 | 只留 50g |
| 加面粉 | 50g |
| 加水 | 50g |
| 等待 | 发酵 2-4 小时 |
| 冷藏 | 存至下周 |

### 健康酵种的标志

| 标志 | 表现 |
|------|------------------|
| 膨胀 | 喂养后 4-6 小时翻倍 |
| 气泡 | 全身充满小气泡 |
| 气味 | 愉悦的微酸味 |
| 质地 | 多孔、蓬松 |

### 在面包中使用酵种

**何时使用：**

- 酵种体积已翻倍
- 通过浮水测试（一小勺浮在水上）
- 喂养后 4-6 小时

**用量：**

| 配方类型 | 酵种用量 |
|-------------|----------------|
| 淡酸味 | 面粉重量的 10-15% |
| 中等酸味 | 面粉重量的 15-20% |
| 浓酸味 | 面粉重量的 20-30% |

## 常见面包种类

### 白吐司

| 原料 | 用量 |
|------------|--------|
| 中筋面粉 | 500g |
| 水 | 300g (60%) |
| 盐 | 10g (2%) |
| 糖 | 25g (5%) |
| 黄油 | 30g (6%) |
| 即发酵母 | 7g (1.4%) |
| 奶粉 | 15g (3%) |

**方法：**

1. 混合所有原料
2. 揉 10-12 分钟至光滑
3. 一次发酵 1-2 小时至两倍大
4. 整形放入吐司模
5. 最终发酵 45-60 分钟
6. 190°C (375°F) 烤 30-35 分钟

### 法式法棍

| 原料 | 用量 |
|------------|--------|
| 高筋面粉 | 500g |
| 水 | 350g (70%) |
| 盐 | 10g (2%) |
| 即发酵母 | 3g (0.6%) |

**方法：**

1. 混合原料，水合 30 分钟
2. 每 30 分钟拉伸折叠一次，持续 2 小时
3. 一次发酵 2-3 小时
4. 分成 3 份
5. 预整形，静置 15 分钟
6. 整形为法棍
7. 最终发酵 45-60 分钟
8. 斜割
9. 240°C (475°F) 带蒸汽烤 20-25 分钟

### 酸面团

| 原料 | 用量 |
|------------|--------|
| 高筋面粉 | 450g |
| 全麦面粉 | 50g |
| 水 | 375g (75%) |
| 盐 | 10g (2%) |
| 天然酵种 | 100g (20%) |

**方法：**

1. 面粉和水混合，水合 1 小时
2. 加入酵种和盐，混合均匀
3. 每 30 分钟拉伸折叠一次，持续 2 小时
4. 一次发酵 4-6 小时至膨胀 50-75%
5. 预整形，静置 20 分钟
6. 最终整形，放入发酵篮
7. 冷藏过夜（8-12 小时）
8. 冷面团放入铸铁锅，240°C (475°F) 烘烤
9. 盖盖 20 分钟，揭盖 20 分钟

### 佛卡夏

| 原料 | 用量 |
|------------|--------|
| 高筋面粉 | 500g |
| 水 | 400g (80%) |
| 盐 | 12g (2.4%) |
| 橄榄油 | 50g (10%) |
| 即发酵母 | 5g (1%) |

**方法：**

1. 混合所有原料
2. 每 30 分钟拉伸折叠一次，持续 2 小时
3. 一次发酵 3-4 小时
4. 倒入抹油的烤盘
5. 拉伸面团填满烤盘
6. 用手指戳出凹痕
7. 淋橄榄油，加配料
8. 最终发酵 30 分钟
9. 220°C (425°F) 烤 20-25 分钟

### 披萨面团

| 原料 | 用量 |
|------------|--------|
| 高筋面粉 | 500g |
| 水 | 325g (65%) |
| 盐 | 10g (2%) |
| 橄榄油 | 15g (3%) |
| 即发酵母 | 3g (0.6%) |

**方法：**

1. 混合原料至光滑
2. 一次发酵 2 小时并做拉伸折叠
3. 分成 250g 的面团球
4. 冷藏 24-72 小时以获得最佳风味
5. 整形前回温至室温
6. 手工拉伸，加配料
7. 最高温度（250-300°C）烤 8-12 分钟

## 问题排查

### 常见问题及解决方案

| 问题 | 可能原因 | 解决方案 |
|---------|---------------|----------|
| 面包密实沉重 | 发酵不足 | 延长一次发酵时间 |
| 面包扁平 | 面筋发展不足 | 多揉、多做拉伸折叠 |
| 大孔洞 | 整形不佳 | 改进整形技巧 |
| 外壳苍白 | 烤箱温度低 | 延长预热，使用温度计 |
| 外壳厚硬 | 烘烤过度 | 减少烘烤时间，检查温度 |
| 口感发粘 | 烘烤不足 | 烤至内部温度达 95°C |
| 面包塌陷 | 发酵过度 | 减少发酵时间 |
| 面包不膨胀 | 酵母失效 | 检查酵母新鲜度、水温 |
| 内部组织不均 | 整形不佳 | 练习整形、制造张力 |
| 味道发酸 | 发酵过度 | 减少发酵时间或降低温度 |

### 面团问题

| 问题 | 原因 | 解决方案 |
|---------|-------|-----|
| 太粘 | 水合度高、揉面不足 | 逐渐加面粉、多揉 |
| 太干 | 水合度低 | 逐渐加水 |
| 拉不开 | 面筋太紧 | 静置 15-20 分钟 |
| 整形时破裂 | 面筋发展不足 | 多做拉伸折叠 |
| 滑离台面 | 台面湿 | 使用刮刀、撒少许面粉 |

### 发酵问题

| 问题 | 原因 | 解决方案 |
|---------|-------|----------|
| 膨胀慢 | 温度低 | 找温暖处，使用发酵箱 |
| 膨胀太快 | 太热、酵母太多 | 降低温度或减少酵母 |
| 不膨胀 | 酵母失效、太冷 | 检查酵母、温暖环境 |
| 发酵过度 | 太久、太热 | 缩短时间、降低温度 |

### 烘烤问题

| 问题 | 原因 | 解决方案 |
|---------|-------|----------|
| 底部烤焦 | 位置太低、深色烤盘 | 上移烤架，用浅色烤盘 |
| 整体苍白 | 时间太短、温度低 | 延长时间、提高温度 |
| 侧面开裂 | 割包不够深 | 割深一点 |
| 粘在烤盘上 | 没用烘焙纸、面粉不够 | 用烘焙纸、撒面粉 |
| 内部未熟 | 温度太高、时间太短 | 降低温度、延长时间 |

## 成功技巧

1. **使用厨房秤** - 按重量比按杯量更精确
2. **控制温度** - 面团温度影响一切
3. **要有耐心** - 好面包需要时间
4. **练习整形** - 这是最难掌握的技能
5. **做好记录** - 记录什么有效什么无效
6. **从简单开始** - 先掌握基础配方再尝试复杂的
7. **观察面团而非时钟** - 时间是参考，不是规则
8. **相信过程** - 即使失败的面包通常也好吃
