## 简介

OpenFarm 是一个开源平台，用于分享和发现种植知识。它提供了结构化的种植指南、作物信息和最佳实践数据库，适合各种经验水平的园丁和农民。

### 平台概览

| 功能 | 描述 |
|------|------|
| 种植指南 | 作物种植的分步指导 |
| 作物数据库 | 全面的植物信息 |
| 用户社区 | 共享经验和建议 |
| API 访问 | 编程方式访问数据 |
| 开源 | 透明、社区驱动 |

### 核心理念

OpenFarm 坚信：

- **开放知识**：免费获取种植信息
- **社区驱动**：全球农民的集体智慧
- **实用导向**：可操作、经过验证的建议
- **生物多样性**：支持多样作物品种
- **可持续性**：环境友好的实践

## 种植指南

### 指南结构

每份种植指南包含标准化章节：

| 章节 | 内容 |
|------|------|
| 概述 | 简要描述和目标 |
| 前置条件 | 开始前需要准备什么 |
| 时间线 | 何时种植和收获 |
| 步骤 | 详细指导 |
| 问题排查 | 常见问题和解决方案 |
| 技巧 | 专家建议 |

### 示例：番茄种植指南

**概述：**
番茄是暖季作物，整个夏季都能结果。需要充足阳光、持续浇水和支撑结构。

**前置条件：**

| 要求 | 详情 |
|------|------|
| 阳光 | 6-8 小时直射阳光 |
| 土壤 | 排水良好，pH 6.0-6.8 |
| 间距 | 植株间 24-36 英寸 |
| 支撑 | 笼子、支架或棚架 |
| 气候 | 暖季，无霜 |

**时间线：**

| 阶段 | 时间 |
|------|------|
| 室内育苗 | 末次霜冻前 6-8 周 |
| 移栽 | 末次霜冻后，土温 > 60 deg F |
| 首次结果 | 移栽后 60-80 天 |
| 收获 | 移栽后 80-120 天 |

**步骤：**

1. **育苗**
   - 在育苗盘中填入育苗基质
   - 种子播深 1/4 英寸
   - 保持湿润和温暖（70-80 deg F）
   - 提供 12-16 小时光照

2. **移栽**
   - 炼苗 7-10 天
   - 比在盆中种得更深（埋入的茎会长出根）
   - 种后充分浇水
   - 基部覆盖覆盖物

3. **日常养护**
   - 每周深浇 1-2 英寸水量
   - 无限生长型品种需打侧枝
   - 每 2-3 周施肥
   - 监测病虫害

## 土壤准备

### 土壤检测

种植前检测土壤：

| 检测项 | 理想范围 | 偏低修正 | 偏高修正 |
|--------|----------|----------|----------|
| pH | 6.0-7.0 | 石灰 | 硫磺 |
| 氮 | 20-40 ppm | 堆肥、血粉 | 减少施肥 |
| 磷 | 25-50 ppm | 骨粉 | 减少施肥 |
| 钾 | 150-250 ppm | 草木灰、绿砂 | 水洗 |
| 有机质 | 3-5% | 堆肥 | 不适用 |

### 土壤改良指南

| 改良剂 | 用途 | 施用量 |
|--------|------|--------|
| 堆肥 | 改善结构、增加养分 | 2-4 英寸/年 |
| 沤熟粪肥 | 增加养分、有机质 | 1-2 英寸/年 |
| 石灰 | 提高 pH | 5-10 磅/100 平方英尺 |
| 硫磺 | 降低 pH | 1-2 磅/100 平方英尺 |
| 珍珠岩 | 改善排水 | 混合 10-20% |
| 蛭石 | 改善保水 | 混合 10-20% |
| 泥炭苔 | 酸化、保水 | 混合 20-30% |
| 骨粉 | 增加磷 | 5-10 磅/100 平方英尺 |

### 土壤结构改善

| 问题 | 症状 | 解决方案 |
|------|------|----------|
| 板结 | 排水差、难以挖掘 | 添加有机质、宽叉松土 |
| 黏土 | 湿时粘、干时裂 | 添加石膏、堆肥 |
| 沙土 | 排水过快、养分低 | 添加堆肥、泥炭苔 |
| 酸性土 | pH < 6.0、生长差 | 添加石灰 |
| 碱性土 | pH > 7.5、养分锁定 | 添加硫磺、泥炭苔 |

### 移栽

**炼苗过程：**

| 天 | 曝露 | 时长 |
|----|------|------|
| 1-2 | 遮阴、受保护 | 2-3 小时 |
| 3-4 | 半日照 | 4-5 小时 |
| 5-6 | 全日照、遮风 | 6-8 小时 |
| 7-8 | 全日照、露天 | 全天 |
| 9-10 | 露天过夜（无霜时） | 24 小时 |

**移栽步骤：**

1. 移栽前充分浇湿幼苗
2. 挖比根球大两倍的坑
3. 坑中加入堆肥
4. 轻轻取出植株，保持根球完整
5. 放入坑中，与原深度相同（番茄可更深）
6. 填土，轻轻压实
7. 充分浇水
8. 基部覆盖覆盖物

## 浇水

### 各作物需水量

| 作物 | 需水量 | 频率 | 关键期 |
|------|--------|------|--------|
| 番茄 | 1-2 英寸/周 | 深浇、不频繁 | 果实发育 |
| 辣椒 | 1-1.5 英寸/周 | 持续 | 开花、结果 |
| 生菜 | 1 英寸/周 | 频繁、少量 | 全程 |
| 黄瓜 | 1-2 英寸/周 | 持续 | 果实发育 |
| 豆类 | 1 英寸/周 | 开花期 | 开花、结荚 |
| 根菜 | 1 英寸/周 | 持续 | 根系发育 |

### 浇水方式比较

| 方式 | 效率 | 最佳用途 | 优点 | 缺点 |
|------|------|----------|------|------|
| 滴灌 | 90-95% | 大多数作物 | 高效、减少病害 | 初始成本高 |
| 渗水管 | 80-90% | 行栽、苗床 | 经济、简单 | 精度较低 |
| 喷灌 | 50-70% | 草坪、覆盖作物 | 覆盖广 | 浪费水、病害风险 |
| 人工浇水 | 60-80% | 小面积、容器 | 精准控制 | 耗时 |
| 陶罐 | 90-95% | 容器、小苗床 | 被动、高效 | 范围有限 |

### 缺水和过水症状

| 缺水症状 | 过水症状 |
|----------|----------|
| 萎蔫（尤其下午） | 叶片发黄 |
| 叶缘干燥、焦脆 | 茎变软 |
| 生长缓慢 | 根腐 |
| 落花落果 | 真菌病害 |
| 土壤脱离盆壁 | 土壤持续湿润 |

### 覆盖物保水

| 覆盖物类型 | 厚度 | 优点 | 注意事项 |
|------------|------|------|----------|
| 稻草 | 3-4 英寸 | 经济、可分解 | 可能含种子 |
| 木屑 | 2-3 英寸 | 持久 | 可能锁氮 |
| 落叶 | 2-3 英寸 | 免费、可分解 | 湿时会压实 |
| 草屑 | 1-2 英寸 | 免费、富氮 | 必须干燥、薄层 |
| 堆肥 | 2-3 英寸 | 增加养分 | 较贵 |

## 病虫害防治

### 综合防治（IPM）

IPM 结合多种策略实现有效、可持续的病虫害防治：

```
IPM 金字塔
         +----------+
         | 化学防治 | （最后手段）
         |          |
         +----------+
         | 生物防治 | （天敌）
         |          |
         +----------+
         | 农艺防治 | （预防）
         |          |
         +----------+
         | 物理/    | （屏障、诱捕）
         | 机械防治 |
         +----------+
```
